## Tech Stack
- **Framework**: Astro (fast static site generation)
- **Styling**: Tailwind CSS (mobile-first)
- **Hosting**: Vercel (free SSL, global CDN)
- **Forms**: Web3Forms or Formspree (GDPR compliant)

## Requirements
- [ ] Mobile-first responsive design
- [ ] GDPR cookie consent banner
- [ ] Secure contact form
- [ ] Fast loading (<3s)
- [ ] Tour showcase (10+ tours)
- [ ] Better than competitor (nolimitsadventure.com)

## Pages
1. **Home** - Hero, featured tours, about snippet, CTA
2. **Tours** - All tours grid with filters
3. **Tour Detail** - Individual tour page template
4. **About** - Company story, team
5. **Contact** - Form + map + details
6. **Privacy Policy** - GDPR compliance
7. **Cookie Policy** - GDPR compliance
